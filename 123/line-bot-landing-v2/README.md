# AI理性陪跑研究室 Landing Page v2

## 1. 本機預覽
```bash
npm install
npm run dev
```

## 2. 部署到 Vercel
1. 登入 Vercel
2. 新增專案
3. 上傳這整個資料夾或匯入 Git 倉庫
4. Framework Preset 選 `Vite`
5. 點 `Deploy`

## 3. 之後你最常改的地方
- `src/App.jsx`
  - 主標題、文案、FAQ、按鈕連結
- `src/styles.css`
  - 顏色、間距、字體大小、版面樣式
